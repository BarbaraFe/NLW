# Sistema de Check-in para Eventos Presenciais

Este é um sistema de check-in desenvolvido para eventos presenciais. Ele permite aos organizadores de eventos controlar a presença dos participantes de forma eficiente e precisa.

# Funcionalidades

- Cadastro de Participantes: Os organizadores podem cadastrar os participantes do evento, incluindo nome, e-mail e outras informações relevantes.

- Check-in dos Participantes: Os participantes podem realizar o check-in no evento no momento da chegada, utilizando métodos como código QR, identificação por RFID ou inserção manual do código do ingresso.

- Lista de Participantes: O sistema fornece uma lista atualizada de participantes presentes no evento, facilitando o acompanhamento em tempo real.

# Tecnologias Utilizadas

- HTML
- CSS
- JavaScript
